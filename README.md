### Инструкция по запуску

1. git clone https://github.com/IlyaL2004/work_game.git
2. Открываем Unity Hub
В правом верхнем углу жмём кнопку Add project from disk (Добавить проект с диска).
В проводнике найти папку work_game, которая только что скачалась Git Bash и зайти внутрь неё.
Внутри будет папка Assets, Packages и файлы. Выделяем эту внутреннюю папку work_game (ту, где лежит Assets) и нажми Select Folder (Выбрать папку).
Unity Hub спросит: какою версию редактора использовать? Выберите Unity 6000.4.1f1 (ту, на которой делал игру) и нажми Confirm.
