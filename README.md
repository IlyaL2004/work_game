### Краткое описание

У нас есть база, которая вырабаьывает энергию. Нужно накопить определённое количество энергии, чтобы победить.
Но нам мешают объекты которые могут уничтожить базу, поэтому нужно отбиваться от них


### Механики

1. Механика управления траекторией (Гравитационное отталкивание)
Вместо классического нанесения урона (пули/лазеры) оружие игрока взаимодействует с физикой объектов. Игрок не уничтожает угрозы напрямую, а меняет их вектор движения, сдувая их от защищаемой зоны. При этом сила толчка очень мягкая (эффект "фена"), что требует от игрока постоянного удержания луча на цели, а не просто точечных кликов.
2. Механика слабых/сильных режимов 
В игре есть жесткое правило соответствия: 3 режима пушки vs 3 типа метеоритов. Это заставляет игрока постоянно анализировать цель:
Неправильный подбор: Пушка слабее цели - цель игнорирует выстрел.
Правильный подбор: Цель плавно улетает от базы.
3. Механика фрагментации (Раскол на осколки)
Это механика наказания за избыточную силу. Если игрок применяет режим выше уровнем цели (например, Сильная пушка на Малый/Средний метеорит), происходит раскол. Средний метеорит делится на 3 Малых. Это превращает одну угрозу в несколько мелких, заставляя игрока быстро переключать внимание и тратить время на добивание "мусора", который разлетается по площадке.
4. Механика упругого столкновения 
Метеориты не взрываются при падении на землю. Они ведут себя по законам физики упругих тел: ударяются, теряют часть энергии, подпрыгивают и катятся по площадке.
Метеорит отнимает жизни при первом ударе, но продолжает физически существовать на земле, отскакивая, пока не затормозит или не улетит за пределы карты.
5. Механика угловых траекторий падения
Угрозы летят не по прямым линиям сверху вниз, а по диагонали (целятся в случайную точку в радиусе базы). 
6. Механика Risk/Reward экономики 
Игроку дается выбор, как распорядиться ограниченным ресурсом (Энергией), который выпадает в виде зон на карте:
Чтобы получить энергию, игрок обязан отвлечься от неба и подойти к зоне на земле. Можно потратить накопленную энергию (клавиша 4), чтобы восстановить HP площадки. Можно не тратить энергию на лечение, копить её до 10000 и выиграть уровень. Но при этом риск потерять базу от метеоритов возрастает кратно.







### Инструкция по запуску

1. git clone https://github.com/IlyaL2004/work_game.git
2. Открываем Unity Hub
В правом верхнем углу жмём кнопку Add project from disk (Добавить проект с диска).
В проводнике найти папку work_game, которая только что скачалась Git Bash и зайти внутрь неё.
Внутри будет папка Assets, Packages и файлы. Выделяем эту внутреннюю папку work_game (ту, где лежит Assets) и нажми Select Folder (Выбрать папку).
Unity Hub спросит: какою версию редактора использовать? Выберите Unity 6000.4.1f1 (ту, на которой делал игру) и нажми Confirm.
